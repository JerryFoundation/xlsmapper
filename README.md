
# XlsMapper

XlsMapper is Java Library for mapping Excel sheets to POJO.

* Licensee

Apache License verion 2.0

# Setup

```xml
<dependency>
	<groupId>com.github.mygreen</groupId>
	<artifactId>xlsmapper</artifactId>
	<version>0.5</version>
</dependency>
```
